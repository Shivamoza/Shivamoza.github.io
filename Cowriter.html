All about Project

Research is a fundamental building block of the process of idea development, but most people struggle with documenting part of it. To simplify this process, we have created an application that helps in generating documentation or reports using NLP. As NLP is the engine that powers language translators, voice command systems, and fast real-time summaries of huge amounts of text. It is an efficient technique which can be used generate computerised texts having human emotions. Report writing is prevalent in a variety of fields. In this paper, we introduced a system CoWriter that generate report using language model which is trained to create the report in a sequential manner based on preceding topics. The model is intended to create the whole report in two different formats i.e., IEEE Format and K.J. Somaiya institute of technology (KJSIT) format.
